# Real-Time Fraud Detection System for Financial Transactions 

**Objectives:** 
- To build a system that detects fraudulent transactions in streaming data, using anomaly detection and alerting mechanisms. 
- Process live data feeds, apply models to flag suspicious activities, and integrate notifications via email/SMS. 

**Key Technologies**:
- Scikit-learn for ML models 
- Keras for autoencoders or isolation forest
- Kafka or Redpanda for streaming
- Flask for API
- Docker for deployment

**Datasets:** 
1. [ULB Credit Card Fraud Dataset](https://figshare.com/articles/dataset/creditcard_Dataset/29270873?utm_source=chatgpt.com&file=55234994)
